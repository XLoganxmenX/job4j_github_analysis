# *job4j_github_analysis*

job4j_github_analysis - проект API приложения для анализа данных с GitHub. Релизован на языке Java с использованием фреймворка
Spring Boot. Приложение предлагает удобное API для сохранения репозиториев и регулярно собирает информацию о последних коммитах с GitHub.

### Основной функционал приложения:
-  Сохранение своего репозитория;
-  Просмотр всех сохраненных коммитов и репозиториев;
-  Регулярный сбор актуальных коммитов из GitHub;

**Стек проекта:**
- **Язык**: Java 17
- **Сборка проекта**: Maven 3.9.6
- **Фреймворк**: Spring Boot
- **База данных**: PostgreSQL 16.2
- **Миграции БД**: Liquibase
- **ORM**: Hibernate

**Требования к окружению**: Java 17, Maven 3.8 или выше, PostgreSQL 14 или выше.

**Запуск проекта:**

- Установить требуемое в проекте ПО (JDK, PostgreSQL);
- Создать базу github_analysis и параметрами подключения из application.properties;
- Скачать и распаковать проект;
- Через терминал перейти в папку проекта и запустить его двумя командами : "mvn clean install", затем "mvn spring-boot:run";
